# JuaLuma Marketing Site

Public marketing and legal website (Next.js + React + TypeScript).

## Commands

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Local URL

- `http://localhost:5177` when started through root `docker-compose.yml`

## Structure

- App routes: `frontend-marketing/src/app/`
- Shared UI: `frontend-marketing/src/components/`
- Static assets and legal text files: `frontend-marketing/public/`
