# Mobile Store Listing Package

Repository-driven source for App Store / Play Store listing artifacts.

## Generate package
```bash
./mobile/scripts/prepare-store-listing-package.sh stage .codex-artifacts/mobile-release-stage.env
```

Optional explicit output path:
```bash
./mobile/scripts/prepare-store-listing-package.sh prod .codex-artifacts/mobile-release-prod.env mobile/release/listing/output/prod-manual
```

## Validate package
```bash
./mobile/scripts/validate-store-listing-package.sh mobile/release/listing/output/<env-timestamp>
```

## Output structure
- `ios/metadata.en-US.json`
- `ios/screenshots/en-US/iphone-67/*.png`
- `android/metadata.en-US.json`
- `android/screenshots/en-US/phone/*.png`
- `shared/app-icon-1024.png`
- `submission-checklist.md`

## Notes
- Screenshot files generated by the script are placeholders from the app icon.
- Replace placeholders with release-accurate screenshots before final submission.
