# JuaLuma Support Portal

Support operations frontend (React + Vite). This app is separate from the public marketing site and the authenticated user app.

## Commands

```bash
npm run dev
npm run build
npm run lint
npm run test
```

## Local URL

- `http://localhost:5176` when started through root `docker-compose.yml`

## Structure

- Source: `support-portal/src/`
- Static assets: `support-portal/public/`
- Build output: `support-portal/dist/`
