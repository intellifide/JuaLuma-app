# JuaLuma Frontend App

Main authenticated user application (React + Vite + TypeScript).

## Commands

```bash
npm run dev
npm run build
npm run lint
npm run test
npm run storybook
```

## Local URL

- `http://localhost:5175` when started through root `docker-compose.yml`

## Structure

- Source: `frontend-app/src/`
- Static assets: `frontend-app/public/`
- Storybook config: `frontend-app/.storybook/`
- Capacitor projects: `frontend-app/android/`, `frontend-app/ios/`
